## Type variable or specific type constructor?
1.
```haskell
f :: Num a => a -> b -> Int -> Int
--           [0]  [1]   [2]    [3]
[0] constrained polymorphic (Num)
[1] fully polymorphic
[2] concrete
[3] concrete
```
2.
```haskell
f :: zed -> Zed -> Blah
--          [0]     [1]
[0] concrete
[1] concrete
```
3.
```haskell
f :: Enum b => a -> b -> C
--             [0]  [1] [2]
[0] fully polymorphic
[1] constrained polymorphic (Enum)
[2] concrete
```

4.
```haskell
f :: f -> g -> C
```

### Write a type signature
1.
```haskell
Prelude> functionH (x:_) = x
Prelude> :t functionH
functionH :: [a] -> a
```
2.
```haskell
Prelude> functionC x y = if (x > y) then True else False
Prelude> :t functionC
functionC :: Ord a => a -> a -> Bool
```
3.
```haskell
Prelude> functionS (x, y) = y
Prelude> :t functionS
functionS :: (a, b) -> b
```

### Given a type, write the function
1.
```haskell
i :: a -> a
Prelude> i a = a
```
2.
```haskell
c :: a -> b -> a 
Prelude> c a b = a
```
3. Yes `c` and `c''` are the same
```haskell
c :: b -> a -> b 
Prelude> c'' b a = b
```
4.
```haskell
c' a b = b
Prelude> c' a b = b
```
5.
```haskell
r :: [a] -> [a]
Prelude> r [a] = (++) [a]
```
